SECTION_INHERIT
SECTION_INHERIT

typedef enum _SECTION_INHERIT {




    ViewShare=1,
    ViewUnmap=2


} SECTION_INHERIT, *PSECTION_INHERIT;



Enumeration type SECTION_INHERIT is used in a call to NtMapViewOfSection.


  • ViewShare Created view of Section Object will be also mapped to any created in future process.
  • ViewUnmap Created view will not be inherited by child processes.
