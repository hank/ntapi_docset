NtRequestWaitReplyPort
NtRequestWaitReplyPort

NTSYSAPI 
NTSTATUS
NTAPI


NtRequestWaitReplyPort(



  IN HANDLE               PortHandle,
  IN PLPC_MESSAGE         Request,
  OUT PLPC_MESSAGE        IncomingReply );



NtRequestWaitReplyPort is used typically by client side in LPC connection.
  • PortHandle HANDLE to Port Object.
  • Request Pointer to LPC_MESSAGE buffer contains request data.
  • IncomingReply Pointer to LPC_MESSAGE buffer filled on return with reply from other side.
