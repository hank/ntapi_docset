NTSYSAPI 
BOOLEAN
NTAPI


RtlTimeFieldsToTime(



  IN PTIME_FIELDS         TimeFields,
  OUT PLARGE_INTEGER      Time );



Function RtlTimeFieldsToTime converts user-readable structure TIME_FIELDS to 64-bit integer.


  • TimeFields Pointer to TIME_FIELDS structure containing time to convert.
  • Time Pointer to LARGE_INTEGER receiving converted time.


See also oposite function RtlTimeToTimeFields.
