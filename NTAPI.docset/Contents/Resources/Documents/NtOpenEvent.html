NtOpenEvent
NtOpenEvent

NTSYSAPI 
NTSTATUS
NTAPI


NtOpenEvent(



  OUT PHANDLE             EventHandle,
  IN ACCESS_MASK          DesiredAccess,
  IN POBJECT_ATTRIBUTES   ObjectAttributes );



  • EventHandle Result of call - HANDLE to Event Object.
  • DesiredAccess See NtCreateEvent for possible access rights.
  • ObjectAttributes Must contain valid Event Object name, in NT Objects Namespace.


Only named events can be opened by this function call.
