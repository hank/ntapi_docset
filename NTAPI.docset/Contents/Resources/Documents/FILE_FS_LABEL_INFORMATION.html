FILE_FS_LABEL_INFORMATION
FILE_FS_LABEL_INFORMATION

typedef struct _FILE_FS_LABEL_INFORMATION {



  ULONG                   VolumeLabelLength;
  WCHAR                   VolumeLabel[1];

} FILE_FS_LABEL_INFORMATION, *PFILE_FS_LABEL_INFORMATION;



Structure is an input buffer for NtSetVolumeInformationFile call with information class FileFsLabelInformation.


  • VolumeLabelLength Length of VolumeLabel array, in bytes.
  • VolumeLabel[1] Label for specified volume.
