NtSetDefaultLocale

NTSYSAPI 
NTSTATUS
NTAPI


NtSetDefaultLocale(



  IN BOOLEAN              UserProfile,
  IN LCID                 DefaultLocaleId );



Function NtSetDefaultLocale sets default Kernel or User Mode locale. See Microsoft SDK for detailed description about locale codes.


  • UserProfile If set, function sets UserMode locale. If not, KernelMode locale is modified.
  • DefaultLocaleId Locale to set.
