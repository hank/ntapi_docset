RtlValidateHeap

NTSYSAPI 
BOOLEAN
NTAPI


RtlValidateHeap(



  IN PVOID                HeapHandle,
  IN ULONG                Flags,
  IN PVOID                AddressToValidate OPTIONAL );






HeapHandle


Address of heap to validate.




Flags


See RtlCreateHeap.




AddressToValidate


If specified, only one block is validated. If not, all blocks from heap are validated.
