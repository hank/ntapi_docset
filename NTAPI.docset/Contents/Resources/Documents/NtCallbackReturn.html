NtCallbackReturn(



  IN PVOID                Result OPTIONAL,
  IN ULONG                ResultLength,
  IN NTSTATUS             Status );



Function NtCallbackReturn finish execution of User-Mode callback.


  • Result Pointer to user's allocated buffer with custom data.
  • ResultLength Length of Result buffer, in bytes.
  • Status Callback execution status code.
