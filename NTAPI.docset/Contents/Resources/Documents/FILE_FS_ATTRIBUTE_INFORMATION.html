FILE_FS_ATTRIBUTE_INFORMATION
FILE_FS_ATTRIBUTE_INFORMATION

typedef struct _FILE_FS_ATTRIBUTE_INFORMATION {



  ULONG                   FileSystemAttributes;
  LONG                    MaximumComponentNameLength;
  ULONG                   FileSystemNameLength;
  WCHAR                   FileSystemName[1];

} FILE_FS_ATTRIBUTE_INFORMATION, *PFILE_FS_ATTRIBUTE_INFORMATION;



FILE_FS_ATTRIBUTE_INFORMATION is output buffer in a call to NtQueryVolumeInformationFile function with FileFsAttributeInformation information class.
  • FileSystemAttributes - ??? (0x1F)
  • MaximumComponentNameLength Maximum length of file name on specified device.
  • FileSystemNameLength Length of FileSystemName array, in bytes.
  • FileSystemName[1] Name of File System on specified device (ex. "NTFS").
