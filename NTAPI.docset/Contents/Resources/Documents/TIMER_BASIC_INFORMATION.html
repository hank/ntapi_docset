TIMER_BASIC_INFORMATION
TIMER_BASIC_INFORMATION

typedef struct _TIMER_BASIC_INFORMATION {



  LARGE_INTEGER           RemainingTime;
  BOOLEAN                 TimerState;

} TIMER_BASIC_INFORMATION, *PTIMER_BASIC_INFORMATION;



  • RemainingTime Contains time (in 100ns units) to next timer signal (negative value), or time after last signalization.
  • TimerState If TRUE, timer is signaled.
